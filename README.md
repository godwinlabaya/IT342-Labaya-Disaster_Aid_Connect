# Login & Registration System (Web & Mobile)

A full-stack authentication application featuring a **Spring Boot** backend, a **React.js** web frontend, and **Android Studio (Kotlin)** mobile app. Both platforms share the same database and authentication logic.

## Features
- **User Registration:** Create a new account with username, email, and password.
- **Secure Login:** Authenticate existing users.
- **Dashboard:** Protected route that only logged-in users can access.
- **Logout:** Securely clears session/local storage.
- **Cross-Platform:** Updates on the web are immediately reflected on mobile (and vice versa).

## Tech Stack

### Backend
- **Framework:** Java Spring Boot
- **Build Tool:** Maven
- **Database:** Supabase

### Frontend (Web)
- **Framework:** React.js
- **HTTP Client:** Axios

### Mobile (Android)
- **Language:** Kotlin
- **UI:** XML Layouts
- **HTTP Client:** Retrofit
